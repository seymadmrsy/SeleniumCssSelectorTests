Test Senaryoları
AddRecordTest
- Kullanıcı arayüzü üzerinden kayıt ekleme senaryosu test edilmiştir  
- Eklenen kayıt üzerinde güncelleme işlemi yapılmış  
- Güncelleme sonrası verinin doğruluğu kontrol edilmiştir  

ClickTest
- CSS Selector kullanılarak tıklama işlemleri gerçekleştirilmiştir  
- Tıklama sonrası ekranda gösterilen mesajın doğruluğu test edilmiştir  

Kullanılan Teknolojiler
- Java  
- Selenium WebDriver  
- Maven  
- TestNG
